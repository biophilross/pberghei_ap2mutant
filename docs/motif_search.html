Finally, we can generate background models for these sequences:

cat promoters_1500bps_corrected.fasta | fasta-get-markov -m 4 > promoters_1500bps_corrected.background

Finally, we can search the sequences.

Search sequences

We’ll use FIMO to search the promoter sequences:

fimo --bgfile promoters_1500bps_corrected.background --oc fimo_promoters_1500_genomic --parse-genomic-coord --thresh 1e-4 8mer_motifs.meme promoters_1500bps_corrected.fasta 

fimo --bgfile promoters_1500bps_corrected.background --oc fimo_promoters_1500 --thresh 1e-4 8mer_motifs.meme promoters_1500bps_corrected.fasta
